Description
===========================================

NeoAxis 3D Engine is integrated development environment of 3D projects of any type and complexity. 
The environment is intended for use in such areas as the creation of video games, the development 
of simulators, the development of virtual reality and visualization software. It includes a full 
set of tools for fast and logical development of modern 3D projects.
